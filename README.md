# SeqoneTV
## A technical test for Seq.one
## Based on :
- Angular/cli 
- Nodejs 
- Docker 
- Babel 


###### Dependencies :
- Docker-compose
- Angular/cli
- Nodejs

## For build the project type the following command line

###### First clone the depo 
```
git clone https://github.com/tashikomaaa/SeqoneTV.git
```
```
cd SeqoneTV 
```

###### Now run the build with :
```bash
npm build
```

###### Now run the project 
```bash 
npm start
```
